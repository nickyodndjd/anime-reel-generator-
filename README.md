# anime-reel-generator

Mobile-friendly Anime Reel Idea Generator built with React + Vite + Tailwind.

## Quick usage
1. Run `npm install` then `npm run dev` to test locally.
2. Deploy to Vercel: connect GitHub repo and deploy (auto-detect Vite/React).

Note: If you want OpenAI integration, provide API key in the UI (not recommended to keep in-browser for security).
